Quick Reference

File System

  • ls - list files
  • cd <dir> - change directory
  • mkdir <dir> - create directory
  • touch <file> - create file
  • cat <file> - view file
  • rm <file> - remove file
  • cp <src> <dest> - copy
  • mv <src> <dest> - move

Git Basics

  • git init - initialize repo
  • git add <file> - stage changes
  • git commit -m "msg" - commit
  • git status - check status
  • git log - view history
  • git diff - see changes

Git Branches

  • git branch - list branches
  • git branch <name> - create
  • git checkout <br> - switch
  • git checkout -b <br> - create & switch
  • git merge <br> - merge
  • git branch -d <br> - delete

Vi Editor

  • vi <file> - open editor
  • i - insert mode
  • Esc - normal mode
  • :w - save
  • :q - quit
  • :wq - save & quit
  • dd - delete line
  • hjkl - move cursor
